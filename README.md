# SHOPIX 🛍️

## AI-Powered Smart Receipt & Shopping Assistant

SHOPIX is a smart web application that helps users organize shopping receipts, track expenses, manage warranties, receive reminder notifications, and view spending analytics.

### Features
- Upload Receipt (Image/PDF)
- OCR Text Extraction
- Expense Dashboard
- Warranty Tracker
- Reminder Notifications
- Offline Support (PWA)
- Light & Dark Theme
- Search & Filter
- Monthly Analytics
